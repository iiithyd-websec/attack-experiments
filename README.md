# Attack-Experiments

## Demonstration steps
- Install npm/node
- run `make all`
- run `cd bag-of-tricks`
- run `node server.js` to start client server locally.
- go to `http://localhost:3100`
- run `cd can-of-worms`
- run `node server.js` to start attacker server locally.
- go to `http://localhost:3200`
